# 📊 Sales Performance Analysis

## 📌 Project Overview

This project analyzes the Sample Superstore dataset using Python. The objective is to identify sales trends, profit performance, and business opportunities through data analysis and visualization.



## 🎯 Objectives

- Clean and prepare the dataset
- Analyze sales and profit
- Identify top-performing regions and categories
- Study the impact of discounts on profit
- Provide business recommendations



## 🛠️ Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook



## 📂 Dataset

Dataset: Sample Superstore Dataset



## 📈 Analysis Performed

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Sales Trend Analysis
- Region-wise Sales Analysis
- Category-wise Sales Analysis
- Profit Analysis
- Discount vs Profit Analysis
- Correlation Analysis



## 🔍 Key Insights

- West region generated the highest sales.
- Technology category contributed the highest revenue.
- Higher discounts often reduced profit.
- Some product categories generated lower profits despite good sales.



## 💡 Business Recommendations

- Reduce excessive discounts.
- Focus on profitable product categories.
- Improve sales strategies in low-performing regions.
- Monitor profit along with sales.
